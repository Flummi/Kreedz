### Deployment
- Clone repository.
- Extract compiler executable and includes to _"compiler"_ folder of project.
- Install dependencies `npm i`

#### Customize builder
Use `config.user.js` file (Generated automatically on dependencies install)

#### Build project

```bash
npm run build
```

#### Watch project

```bash
npm run watch
```

#### Create bundle

```bash
npm run pack
```
